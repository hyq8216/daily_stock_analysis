# 选股策略更新与回测系统

## 📊 策略改进说明

### 原策略问题
1. **单一维度**：仅基于连续小阳线形态
2. **缺乏验证**：没有历史回测验证策略有效性
3. **参数固定**：无法根据市场情况调整

### 新策略特点

#### 1. 多维度评分系统
| 维度 | 权重 | 说明 |
|------|------|------|
| 形态得分 | 40% | 小阳线连续性、涨幅合理性 |
| 成交量得分 | 30% | 量比、放量程度 |
| 趋势得分 | 20% | 均线排列、价格位置 |
| 动量得分 | 10% | 短期涨幅、动能强度 |

#### 2. 可配置参数
所有策略参数已移至 `strategy_config.py`，可根据市场情况调整：
- 连续天数要求
- 涨跌幅限制
- 成交量要求
- 均线条件
- 回测周期
- 止损止盈

#### 3. 完整回测系统
`backtester.py` 提供：
- 历史数据回测
- 胜率统计
- 收益率计算
- 止损止盈模拟
- 交易成本扣除

## 🔧 使用方法

### 运行改进版选股
```bash
python src/advanced_strategy.py
```

### 运行策略回测
```bash
python src/backtester.py
```

### 调整策略参数
编辑 `src/strategy_config.py` 中的配置项

## 📈 回测指标说明

| 指标 | 说明 |
|------|------|
| 胜率 | 盈利交易次数/总交易次数 |
| 平均收益 | 单笔交易平均收益率 |
| 总收益 | 所有交易累计收益率 |
| 最大盈利 | 单笔最大收益率 |
| 最大亏损 | 单笔最大亏损率 |
