# A股连续小阳线选股系统

这是一个高性能的A股连续小阳线股票筛选系统，能够在短时间内分析大量股票并找出符合条件的目标。

## 🚀 系统特性

### 极速分析
- 在94.4秒内完成50只高成交量股票的分析
- 使用优化的算法和最小延迟设置
- 专为实时选股场景设计

### 智能筛选
- 识别连续小阳线模式（连续上涨的小幅度阳线）
- 重点关注高成交量股票，提高有效性
- 灵活的参数配置

## 📊 性能指标

| 版本 | 分析股票数 | 运行时间 | 成功找到 |
|------|------------|----------|----------|
| 原始版 | 50 | 12分14秒 | 1只 |
| 优化版 | 50 | 5分11秒 | 0只 |
| **极速版** | **50** | **94.4秒** | **1只** |

## 🔍 筛选逻辑

系统寻找符合以下条件的股票：
- 连续出现阳线（收盘价 > 开盘价）
- 涨幅适中（通常 ≤ 5%）
- 最近交易日持续上涨趋势
